Instruções de Implantação - Centro de Treinamento

Visão Geral

Este documento fornece instruções para implantar o site do Centro de Treinamento na Vercel. O site foi desenvolvido como uma vitrine de produtos com potencial para expansão futura para um e-commerce completo.

Conteúdo do Pacote

O pacote contém os seguintes componentes:

  • Arquivos HTML - Páginas principais e templates
  • Arquivos CSS - Estilos e responsividade
  • Arquivos JavaScript - Funcionalidades interativas e dados de produtos
  • Pasta de imagens - Imagens placeholder para produtos e elementos visuais

Requisitos

  • Conta na Vercel
  • Git instalado (opcional, para método de implantação via Git)

Métodos de Implantação

Método 1: Implantação Direta via Vercel

  1. Faça login na sua conta da Vercel
  2. Clique em "Add New..." e selecione "Project"
  3. Selecione "Upload" na seção "Import Git Repository"
  4. Arraste e solte a pasta do projeto ou clique para selecionar o arquivo ZIP
  5. Configure o nome do projeto e outras opções conforme necessário
  6. Clique em "Deploy" para iniciar a implantação

Método 2: Implantação via Git (Recomendado)

  1. Crie um novo repositório no GitHub, GitLab ou Bitbucket
  2. Inicialize o Git no diretório do projeto: 
    git init
git add .
git commit -m "Versão inicial do site do Centro de Treinamento"
  3. Adicione o repositório remoto: 
    git remote add origin [URL_DO_SEU_REPOSITÓRIO]
git push -u origin main
  4. Faça login na sua conta da Vercel
  5. Clique em "Add New..." e selecione "Project"
  6. Importe o repositório que você acabou de criar
  7. Configure o nome do projeto e outras opções conforme necessário
  8. Clique em "Deploy" para iniciar a implantação

Nota: A implantação via Git é recomendada pois facilita atualizações futuras. Cada vez que você enviar alterações para o repositório, a Vercel automaticamente implantará a nova versão.

Configuração de Domínio Personalizado (Opcional)

  1. Na Vercel, acesse o projeto implantado
  2. Vá para a aba "Settings" e depois "Domains"
  3. Adicione seu domínio personalizado e siga as instruções para configurar os registros DNS

Gerando Imagens Placeholder

O site inclui um gerador de imagens placeholder para facilitar o desenvolvimento:

  1. Após a implantação, acesse [SEU_DOMÍNIO]/gerar-imagens.html
  2. Clique no botão "Gerar Imagens Placeholder"
  3. As imagens serão geradas e baixadas automaticamente
  4. Substitua essas imagens por imagens reais quando disponíveis

Importante: Para um site em produção, substitua as imagens placeholder por imagens reais de alta qualidade. As imagens placeholder são apenas para fins de desenvolvimento.

Expandindo para E-commerce

O site foi projetado para ser facilmente expandido para um e-commerce completo no futuro. Para isso, você precisará:

  1. Implementar a integração com o PagBank conforme o tutorial fornecido
  2. Adicionar funcionalidades de carrinho de compras
  3. Implementar sistema de gerenciamento de pedidos
  4. Configurar métodos de envio
